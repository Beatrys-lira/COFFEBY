# ☕ COFFEBY

Site de cafeteria desenvolvido com foco em design atrativo e experiência do usuário.

## 🖥️ Demonstração

> Em breve — deploy disponível futuramente.

## ✨ Funcionalidades

- Apresentação de produtos e cardápio
- Design elegante inspirado na cultura do café
- Layout responsivo para diferentes dispositivos

## 🛠️ Tecnologias

- **HTML5** — estrutura e semântica
- **CSS3** — estilização, animações e responsividade

## 📁 Estrutura do Projeto

```
COFFEBY/
```

## 🚀 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/Beatrys-lira/COFFEBY.git
   ```
2. Abra o arquivo `index.html` no seu navegador.

> Não requer instalação de dependências.
